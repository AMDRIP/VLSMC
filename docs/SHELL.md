Здесь идеи для новых команд в Shell


Системные:
1. ata - информация о ATA устройствах
2. ahci - информация о AHCI устройствах
3. pci - информация о PCI устройствах
4. usb - информация о USB устройствах
5. exit - выход из системы
6. reboot - перезагрузка системы
7. shutdown - выключение системы

Пользовательские
```markdown
1. ls - список файлов и папок
2. cd - смена текущей директории
3. cat - чтение содержимого файла
4. mkdir - создание новой директории
5. rm - удаление файла или директории
6. ps - список активных процессов
7. kill - завершение процесса
8. clear - очистка экрана терминала
9. help - вывод справки по командам
10. whoami - имя текущего пользователя
```
